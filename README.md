# SteamGuardLite

Lightweight WPF utility that displays Steam Guard codes from a Steam `.maFile`.

## Features
- Load `.maFile` via menu
- Drag & drop `.maFile` onto the window
- Displays the current Steam Guard code
- Auto-refresh every 30 seconds
- One-click copy to clipboard
- Thin progress bar under the code button (shows time until refresh)

## Usage
1. Click **Load** and select your `.maFile` (or drag & drop it into the window).
2. Click the code button to copy the current code.

## Download
Go to **Releases** and download the latest `SteamGuardLite-*-win-x64.zip`.
